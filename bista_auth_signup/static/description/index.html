Signup with Address, Phone, Date of Birth and Captcha

While signup default odoo have only email, username and password. After installing this you can able to get the complete address with states and country along phone number

  • New Customer can give complete address with phone number while signup.
  • List of all States and Countries will be shown selected field in Signup page.
  • Add Date of Birth field.
  • You can choose whether Phone/Address/Date of Birth need to shown in Signup page in Configuration. Default will be true.
  • Upload profile picture while sign up(Upcoming feature)
  • Enable/Disable Captcha in both Signup and Login page.
